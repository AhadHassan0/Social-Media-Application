# Social Media Application - OOP Project

A comprehensive social media desktop application developed using **C++** and the **Qt Framework**. This project was designed to demonstrate the practical application of **Object-Oriented Programming (OOP)** principles in building a scalable, interactive, and user-centric software system.

## 🚀 Key Features

* **User Management:** Create and manage unique user accounts with persistent data storage.
* **Post & Page System:** * **Posts:** Users can create and share updates on their personal timelines.
    * **Pages:** Functionality to create and follow dedicated community or business pages.
* **Interaction Engine:** * **Likes:** Integrated feedback system to interact with content.
    * **Comments:** Ability to add and view comments on posts and pages.
* **Date & Timeline Feature:** Every activity is timestamped, ensuring a chronological flow of data and posts.
* **Activity Page:** A centralized dashboard where users can track their historical interactions, including past likes, comments, and posts.

## 🛠 Technical Implementation

This project implements the core pillars of OOP:

* **Encapsulation:** Protecting user data and state through private class members and controlled public interfaces.
* **Inheritance:** Utilizing a hierarchical structure for content types (e.g., specific classes for Posts and Pages inheriting from a base Content class).
* **Polymorphism:** Using virtual functions and Qt's signal-and-slot mechanism for dynamic UI updates.
* **Abstraction:** Decoupling the backend logic (logic handling and data management) from the frontend GUI.

## 💻 Tech Stack

* **Language:** C++ (Standard 17+)
* **Framework:** Qt 6 (Widgets)
* **Build System:** CMake / qmake
* **Platform:** Windows/Linux/macOS (Cross-platform support via Qt)

## ⚙️ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/AhadHassan0/Social-Media-Application.git](https://github.com/AhadHassan0/Social-Media-Application.git)
    ```
2.  **Open in Qt Creator:**
    * Launch Qt Creator.
    * Open the `CMakeLists.txt` file.
3.  **Build and Run:**
    * Select your compiler kit (MinGW).
    * Click the **Run** button to compile and launch the application.
